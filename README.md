# 贪吃蛇游戏

## 项目简介
这是一个基于原生 HTML、CSS 和 JavaScript 实现的经典贪吃蛇游戏。玩家可以通过键盘控制蛇的移动，目标是吃掉食物并尽可能长地生存下去。游戏提供了简单、中等和困难三种难度。

## 技术栈
- HTML
- CSS
- JavaScript

## 如何启动项目
1. 克隆或下载项目代码到本地。
2. 打开项目文件夹，找到 `index.html` 文件。
3. 使用浏览器打开 `index.html` 文件即可开始游戏。

## 游戏说明
- 使用方向键控制蛇的移动方向。
- 吃掉食物可以增加得分和蛇的长度。
- 蛇撞到墙壁、障碍物或自身时游戏结束。
- 游戏结束后可以输入用户名保存得分，并在排行榜中查看。
- 排行榜按游戏难度分为简单、中等和困难三个部分。
